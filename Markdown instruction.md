## Спсики

Чтобы добавить ненумеованные списки, необходимо пункты  выделить звездочкой(*). Напривер, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумернованные списки, необходимо пункты просто пронумеровать. Напривер, вот так:

1. Первый пункт
2. Второй пункт
3. Третий пункт
